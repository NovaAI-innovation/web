# Chimera Enterprise Logo System (Production Draft)

## Primary System
- Primary lockup: `chimera-primary-on-dark.svg` and `chimera-primary-on-light.svg`
- Horizontal lockup: `chimera-horizontal-on-dark.svg`
- Icon-only mark: `chimera-icon-ce-on-dark.svg`

## One-Color Production
- Black: `chimera-primary-1color-black.svg`
- White: `chimera-primary-1color-white.svg`

## Color Values
- Chimera Black: `#111111`
- Charcoal: `#1A1A1A`
- Steel Gray: `#3A3F45`
- Safety Gold: `#C79A2C`
- White: `#F5F5F5`

## Usage Rules
- Use one-color assets for embroidery, etching, stamps, and low-fidelity print.
- Do not apply gradients, shadows, bevels, or glows.
- Maintain clear space equal to the width of the `C` vertical stem around the lockup.
- Minimum width:
  - Primary lockup: 120 px digital / 30 mm print
  - Horizontal lockup: 160 px digital / 40 mm print
  - Icon-only: 24 px digital / 8 mm print

## Recommended Deployment
1. Website header: `chimera-horizontal-on-dark.svg`
2. Favicon/app icon: `chimera-icon-ce-on-dark.svg` (create png exports at 512/192/32)
3. Vehicle/signage: `chimera-primary-1color-white.svg` on dark wraps, `chimera-primary-1color-black.svg` on light substrates
